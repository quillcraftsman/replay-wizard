Install
-------

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install replay-wizard

From release page
^^^^^^^^^^^^^^^^^

Download source code from `GitHub Releases page <https://github.com/quillcraftsman/replay-wizard/releases>`_

Clone from GitHub
^^^^^^^^^^^^^^^^^

.. code-block:: bash

   git clone https://github.com/quillcraftsman/replay-wizard.git
   make install
